4.3.2 Evaluasi Akurasi Model Peramalan

Setelah memperoleh hasil peramalan menggunakan metode Holt-Winters, tahap berikutnya adalah melakukan evaluasi untuk mengetahui tingkat akurasi model. Evaluasi ini bertujuan mengukur seberapa dekat nilai forecast dengan data aktual. Pengukuran akurasi dilakukan menggunakan beberapa indikator, yaitu Error, Absolute Error, APE (Absolute Percentage Error), dan MAPE (Mean Absolute Percentage Error) pada periode testing.

Rumus Perhitungan Akurasi

1. Error
Error = Y‚Çú - F‚Çú
Mengukur selisih antara data aktual dan hasil peramalan
2. Absolute Error
Absolute Error = |Y‚Çú - F‚Çú|
Menunjukkan besar kesalahan tanpa memperhatikan arah
3. APE (Absolute Percentage Error)
APE = (|Y‚Çú - F‚Çú| / Y‚Çú) √ó 100%
Persentase kesalahan terhadap nilai aktual
4. MAPE (Mean Absolute Percentage Error)
MAPE = Œ£ APE / Jumlah Periode Testing
Rata-rata seluruh nilai APE pada periode testing
